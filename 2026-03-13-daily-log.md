What I learned
Variables in C live in memory and each one has an address. `&` lets you see that address.

What confused me
At first I didn’t get why addresses matter when we can just use the variables.

How it became clearer
C lets you mess with memory directly, that’s why addresses are important for pointers.

Reflection
Even a simple variable is actually somewhere in memory. Seeing this makes programming feel closer to what the computer really does.
